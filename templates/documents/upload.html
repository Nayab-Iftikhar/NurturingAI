How It Works
  1. Upload: Select a brochure file (PDF, TXT, DOCX, or DOC)
  2. Extract: Text is extracted from the document
  3. Chunk: Text is split into manageable chunks (1000 chars with 200 char overlap)
  4. Embed: Each chunk is converted to a vector using sentence-transformers
  5. Store: Vectors, chunks, and metadata are stored in ChromaDB
  6. Search: Documents can be searched semantically for RAG queries
